# Aurelle — Shopify Marketplace Connect Store

A premium, ready-to-sell **skincare** Shopify store built as a custom **Liquid theme**
(Online Store 2.0) for the Digital Heroes 2-day challenge.

- **Brand:** Aurelle — clean, effective skincare
- **Theme:** custom Liquid theme in [`theme/`](theme/) (Dawn-style OS 2.0 structure)
- **Design system:** 60/30/10 palette (ivory · sage/cream · terracotta), Fraunces + Jost
- **Apps integrated:** Judge.me · Shopify Bundles · Subscriptions · Search & Discovery · Marketplace Connect · Labeler

## Repo layout

```
theme/                 # the uploadable Shopify theme (Liquid, OS 2.0)
  assets/              # base.css design system, global.js (cart/gallery/variants)
  config/              # settings_schema.json + settings_data.json (brand defaults)
  layout/              # theme.liquid, password.liquid
  locales/             # en.default.json
  sections/            # hero, featured-collections, benefits, bundle, testimonials, main-product, …
  snippets/            # product-card, price, rating-stars, badge, icon, cart-drawer
  templates/           # index/product/collection/cart/page.*/search/404 (+ customers/)
products.csv           # Shopify product import (5 products + bundle)
brand/images/          # generated product/collection/hero/logo/favicon PNGs
scripts/gen-images.mjs # regenerates brand imagery (sharp)
scripts/seed-store.mjs # creates products+images, collections, pages, menus via Admin API
scripts/config.mjs     # zod-validated Shopify credentials (loads .env)
SETUP.md               # ← what to configure inside Shopify (read this)
agent.md               # architecture & conventions
```

## Setup (pnpm)

```bash
pnpm install            # installs Shopify CLI + sharp (build scripts pre-approved)
pnpm gen:images         # (re)generate brand imagery into brand/images/
pnpm login               # = shopify login --store fetchwell-edyplil9.myshopify.com
pnpm dev                # live theme preview  (shopify theme dev --path theme)
pnpm push               # upload theme        (shopify theme push --path theme)
pnpm check              # theme-check linter   (currently 0 errors)
pnpm package            # zip the theme for manual upload
```

### Seed store data (products, collections, pages, menus)

Populates the store via the Shopify Admin API — no manual CSV/clicking needed.

```bash
# .env (gitignored) — credentials are validated by scripts/config.mjs (zod)
#   SHOPIFY_STORE=fetchwell-edyplil9.myshopify.com
#   SHOPIFY_ADMIN_TOKEN=shpat_xxxxxxxx        # custom app → Admin API access token
pnpm seed               # idempotent: creates/enriches products+images, collections, pages, menus
```

Required Admin API scopes: `write_products, read_products, write_publications, read_publications, write_content, write_online_store_navigation`.
